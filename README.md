Zombies panic
=============

It's a 2D video game with zombies, the goal is kill all zombies. The actual version is a base for a awesome game, so wait'n'see ;)

Installation
------------

```
$ npm install
```

Start game
----------

Start a http server

```
$ npm install -g http-server
$ http-server .
```

Go to `http://localhost:8080`

Enjoy !
